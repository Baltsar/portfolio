monero-mcp — case study
MONERO
MCP
World's First · Open Source · Model Context Protocol

World's First Monero MCP Server

Identifierade att ingen hade byggt en Model Context Protocol-server för Monero. Vibecoded i Cursor med Claude som co-pilot. Hela arkitekturen tog en eftermiddag — Claude kommunicerar med Monero-noden via JSON-RPC. mcp-server-architecture.png

Shippade som open source på GitHub. Fungerande — AI-agenter kan nu interagera med Monero-nätverket direkt via MCP. Första demo:n visade en Claude-agent som checkade wallet-balans och skickade en transaktion. mcp-demo-interaction.gif

Bidrog också till Moneros första Android-wallet (FOSS). Designarbete utan hierarki — UI-koncept skickades som GitHub issues, feedback kom via Discord. monero-wallet-ui-concept.png

Agentflödet designades för att vara modulärt — varje MCP-tool gör en sak. Wallet balance, send transaction, get block, verify payment. agent-flow-diagram.png
